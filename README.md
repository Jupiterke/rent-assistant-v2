# Rent Assistant App 🏠

## 📌 Description
Rent Assistant is a simple rental management tool that helps landlords and tenants track rent payments, due dates, and tenant information easily from a mobile or web app.

## 🚀 Features (MVP)
- Add tenants
- Track rent amounts
- Simple payment list
- Lightweight and mobile-friendly

## 🏗️ Future Features
- Rent due reminders
- Payment history tracking
- M-Pesa integration
- Landlord dashboard
- PDF receipts
- Rent score system

## 🧑‍💻 Built With
- HTML
- CSS
- JavaScript
- GitHub Pages (for hosting)

## 📱 Purpose
To simplify rent tracking in small and medium housing setups in Kenya and beyond.
